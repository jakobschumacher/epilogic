Epilogic

Transform decision notation models into formatted documents

Drop your DMN file here or click to browse

Supported formats: .dmn, .xml (max 10MB)

How it works

  1. Create your DMN file with https://demo.bpmn.io/dmn
  2. Add descriptions to your decision elements and input data
  3. Add required metadata in extensionElements:
    • Required: krankheit, erreger, stand, version
    • Optional: inkubationszeit
  4. Add optional input data elements:
    • klinisches_bild, labordiagnostik, epidemiologische_bestaetigung
    • zusatzinformation, referenzdefinition, meldepflicht, uebermittlung
  5. Upload your DMN 1.3 file for RKI Falldefinitionen
  6. The file is validated and parsed
  7. Choose your desired export format:
    • Word - Full RKI Falldefinition document
    • Markdown - Decision tables only
    • C# / Java - Code generation (coming soon)
  8. The document downloads automatically

All processing happens in your browser. No data is sent to any server.
